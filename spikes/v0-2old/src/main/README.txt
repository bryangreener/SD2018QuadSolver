Folder contents:

- main.c
  main method that acts as a central IO/buffer for the project.
  This method does not perform any operations other than controlling program
  flow.

- main.h
  Header file for main method that handles all dependencies for main method.
  This file has #includes for each other method in the program that it
  directly communicates with.

- main.o
  Object file for main method generated by makefile in root

- makefile
  Temporary makefile. Not in use.

- src
  Directory following 'src' directory stucture in root

Folder contents:

- vdouble.c
  vdouble method that validates double inputs a, b, and c (for ax^2+bx+c=0)
  passed in by main.

- vdouble.h
  Header file for vdouble method that includes std libraries for validating
  char arrays.

- vdouble.o
  Object file generated by makefile in root.

- vmenu.c
  vmenu method that validates int input for menu options passed in by main.

- vmenu.h
  Header file for vmenu method that includes std libraries for validating
  char arrays.

- vmenu.o
  Object file generated by makefile in root.

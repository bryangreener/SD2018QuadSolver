Folder contents:

- wout.c
  wout method that contains multiple functions whose only task is to use the
  printf function for console output. Functions take in values passed in by main
  and get string formatted for output.

- wout.h
  Header file for wout method that includes string and std libraries for
  string formatting and console output.

- wout.o
  Object file generated by makefile in root.
